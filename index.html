Description

Memory Game is a simple but fun game, which by playing it can help improve your memory. The aim of the game is to have matched all the pairs in the fewest amount of moves possible. You make a match by clicking on one of the empty cells to reveal what's underneath, then selecting another cell that has the same color. It's unlikely that your first move will be a match, so the cards will turn back around and allow you to make another selection. Try to remember which colour the circles were that you picked before, as you will need to match them later!

You can click the reveal button to reveal the rest of the board that hasn't yet been matched. You can press new game in order to start again.
